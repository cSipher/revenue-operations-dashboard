# Revenue Operations Dashboard

## Project Overview
Interactive Excel dashboard built for analyzing revenue operations and sales performance.

## Features
- KPI Cards
- Monthly Revenue Trend
- Revenue by Product
- Revenue by Region
- Deal Conversion Breakdown
- Top Sales Agents Analysis
- Interactive Slicers
- PivotTables & PivotCharts

## Tools Used
- Microsoft Excel
- PivotTables
- PivotCharts
- Slicers
- Data Visualization
